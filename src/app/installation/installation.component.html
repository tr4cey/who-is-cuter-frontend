Angular Installation
  1. Download Node.js, you can download from nodejs.org
  2. Make sure you have the npm client installed, enter the command: npm -v in terminal
  3. Install the Angular CLI, you use npm to do this, enter the command: npm install -g @angular/cli
  4. To create a workspace, enter the command: ng new my-app

Laravel Installation
  1. Laravel has a few server requirments, so it is recommended to have the Laravel Homestead virtual machine
  2. Before installing Laravel, you need to download Composer. Use this link: https://getcomposer.org/download/
  3. Next enter this command into the terminal: composer global require laravel/installer
  4. Then you want to create a directory containing a Laravel installation with all of the dependencies. To do this enter the command: laravel new nameofdirectory (nameofdirectory is whatever you choose to name the directory)
  5. To use PHP's built-in development server enter the command: php artisan serve
